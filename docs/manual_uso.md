# Manual de Uso - CentroPlus Connect

Guía visual y práctica para navegar por la aplicación.

Al iniciar la aplicación, se presenta el Dashboard principal.

### Resumen de Estado
En la parte central verás tarjetas con el conteo de:
- **Actividades**: Total de cursos disponibles.
- **Reservas**: Tus inscripciones activas.
- **Incidencias**: Estado de tus comunicaciones.

## 2. Explorar Actividades
Accede mediante el icono 🎯.

```text
Estructura de la Tarjeta de Actividad:
+-----------------------------------+
| Nombre de la Actividad            | <- Título
| TIPO DE ACTIVIDAD                 | <- Subtítulo (Indigo)
| 60 min · 25.50 € · 7 libres       | <- Info secundaria
+-----------------------------------+
```

- **Scroll Fluido**: Desliza hacia arriba o abajo. Notarás un **desvanecimiento (fade)** en los bordes para una experiencia más inmersiva.
- **Selección**: Al pulsar una actividad, el borde se resaltará suavemente.

## 3. Reportar una Incidencia
Accede mediante el icono ⚠️.

1. **Formulario**: Escribe el asunto y la descripción del problema.
2. **Envío**: Pulsa "Enviar Incidencia". La app procesará la solicitud con una animación de pulsación.
3. **Seguimiento**: Tu incidencia aparecerá inmediatamente en la lista inferior con el estado "ABIERTA".

## 4. Barra de Navegación Inferior
Iconos rápidos para una navegación eficiente:

| Icono | Sección | Descripción |
| :---: | :--- | :--- |
| 🏠 | **Inicio** | Resumen y acceso rápido. |
| 🎯 | **Actividades** | Catálogo completo. |
| 📅 | **Reservas** | Mis inscripciones. |
| ⚠️ | **Incidencias** | Soporte técnico. |

---
*Nota: Si dejas el cursor sobre un icono, verás una etiqueta (Tooltip) con el nombre de la sección.*
